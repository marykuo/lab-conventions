## 分支命名 Branch Naming

依據用途：

- `main` 分支：穩定的生產環境版本。
- `develop` 分支：開發中的最新變更。
- `release/<版本號>` 分支：準備發布的版本。

依據目的：

- `feature/<功能名稱>` 分支：新功能開發。
- `bugfix/<修正名稱>` 分支：錯誤修正。
